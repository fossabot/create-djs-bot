# create-djs-bot
디스코드 봇 프로젝트를 만드는 CLI 도구입니다  
타입스크립트도 지원합니다


## install
### npm
```
$ npm install -g @sujang/create-djs-bot
```
### yarn
```
$ yarn global add @sujang/create-djs-bot
```


## Usage
```
$ create-djs-bot <name>
```
<name> 으로 디스코드 봇 프로젝트를 만듭니다


## Example
```
$ create-djs-bot super-bot
```
